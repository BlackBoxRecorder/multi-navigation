# MultiNavigation 网站标题 & 帮助弹窗设计

## 概述

为 MultiNavigation 网站左侧面板增加品牌标题栏，并提供标签页形式的帮助弹窗，帮助新用户了解功能和使用方法。

## 一、网站标题栏

### 位置
左侧面板 (`#leftPanel`) 顶部，位于「我的地点」标题上方。

### 布局
标题栏采用 flex 行布局，左右两端对齐：
- **左侧**：网站名「MultiNavigation」，搭配地图图标（🗺️ emoji），字号 `text-lg font-bold`，品牌色（蓝紫色渐变）或深灰色 `text-gray-800`
- **右侧**：帮助按钮，圆形 `?` 图标，`w-7 h-7`，hover 时背景变色，cursor: pointer

### 与下方内容间距
标题栏与「我的地点」之间留 `mt-3` 或 `mb-2` 间距，视觉上形成 section 分组。

### 页面 title 同步修改
`<title>` 标签从「高德地图多点规划工具」改为「MultiNavigation - 多点路径规划」。

---

## 二、帮助弹窗

### 触发方式
- 点击标题栏右侧 `?` 按钮

### 关闭方式
- 点击弹窗右上角 ✕ 按钮
- 点击弹窗外的遮罩背景
- 按 ESC 键

### 弹窗结构
全屏居中遮罩层，弹窗主体宽约 `560px`，最大高度 `80vh`，超出内容可滚动。

```
┌──────────────────────────────────────┐
│  MultiNavigation 使用帮助               ✕   │  ← 标题栏
├──────────────────────────────────────┤
│  [功能概述] [操作指南] [视频教程]     │  ← 标签页
├──────────────────────────────────────┤
│                                      │
│  (当前标签页内容)                    │
│                                      │
└──────────────────────────────────────┘
```

### 标签页内容

#### 标签页 1：功能概述
介绍 MultiNavigation 四大核心能力：
1. **🗺️ 地图搜索浏览** — 使用顶部搜索栏搜索城市、地点、景区，地图自动定位并展示 POI 标记
2. **📍 地点收藏管理** — 右键点击地图 POI 即可收藏到「我的地点」列表；支持勾选、删除、清空，数据自动保存到浏览器本地
3. **🚗 多模式路线规划** — 勾选出发地 + 点击目的地 POI，自动计算驾车/公交/步行/骑行四种路线，支持切换交通模式查看具体方案
4. **⭐ 多点最优路径** — 右侧面板「计算最优路径」按钮，利用高德路径规划 API 计算途经多点的最优行车路线（最多 16 个地点）

#### 标签页 2：操作指南
分步骤说明：
1. **添加地点** — 在顶部搜索栏输入地点名并搜索，或在地图上直接右键点击 POI 标记 → 选择「收藏到我的地点」
2. **选择出发地** — 在左侧「我的地点」列表中，勾选一个或多个复选框作为出发地（至少勾选 1 个，最多 5 个）
3. **设置目的地** — 在地图上点击任意 POI 标记，弹出信息窗，点击「设为目的地」按钮
4. **查看路线** — 右侧「路径规划」面板自动显示驾车路线方案，可切换顶部「驾车/公交/步行/骑行」标签查看不同交通模式
5. **多点最优路径** — 如需按顺序途经多个地点，点击右侧面板「计算最优路径」按钮，系统自动规划多点间最优路线
6. **清空地点** — 点击左侧面板底部「清空所有地点」按钮可一键清除所有收藏

#### 标签页 3：视频教程
- 视频占位区域：带虚线边框的矩形区域（宽 `480px`，高 `270px`），居中显示
- 占位文字：「📹 视频教程即将上线，敬请期待」
- 后续替换方式：将占位 `<div>` 替换为 `<video>` 标签，指向视频文件路径即可

## 三、实现要点

### 涉及文件
| 文件 | 改动 |
|------|------|
| `index.html` | 新增标题栏 HTML、帮助弹窗 HTML（包括标签页结构）；修改 `<title>` |
| `src/ui.js` | 新增帮助弹窗的打开/关闭/切换标签页事件绑定 |

### 弹窗交互逻辑（ui.js 新增）
- 绑定 `?` 按钮 click → 显示弹窗（移除 `hidden` class）
- 绑定 ✕ 按钮 / 遮罩 click / ESC 键 → 隐藏弹窗（添加 `hidden` class）
- 绑定标签页按钮 click → 切换对应内容面板显示（通过 data 属性关联）

### 样式
- 全部使用 Tailwind CSS 工具类，无额外 CSS 文件
- 弹窗遮罩：`fixed inset-0 bg-black/50 z-50 flex items-center justify-center`
- 弹窗主体：`bg-white rounded-xl shadow-2xl w-[560px] max-h-[80vh] overflow-hidden flex flex-col`
- 标签页激活态：底部蓝色下划线 + 文字颜色变化
- 视频占位区：`border-2 border-dashed border-gray-300 rounded-lg bg-gray-50`

### 不涉及
- 不新增 JS 模块文件（弹窗逻辑归入 ui.js）
- 不修改其他现有模块（map.js / location.js / route.js / utils.js）
- 不修改路由、收藏等业务逻辑
